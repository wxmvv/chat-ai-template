# chat-ai-template

A minimal AI chat interface inspired by ChatGPT.

This project provides a lightweight, dependency-free implementation of a streaming AI chat UI that can be copied and used directly in any project. It focuses on simplicity, portability, and ease of integration.

## ✨ Features

- 💬 ChatGPT-style conversation interface
- ⚡ Streaming response support (real-time output)
- 🧩 No external dependencies required
- 🪶 Minimal CSS and JavaScript footprint
- 📦 Copy-paste ready — works even inside large projects
- 🔧 Easy API configuration
- 🧱 Multiple implementations included:
  - React version
  - Vue version
- 🤖 Uses Ollama API by default

## 🎯 Design Goals

This template is built with the following principles:

- **Minimalism** — Keep code simple and readable
- **Portability** — Drop into any project without setup
- **Framework Flexibility** — Works with or without frameworks
- **Streaming First** — Designed for real-time AI output
- **Zero Lock-in** — API and UI are easy to modify
